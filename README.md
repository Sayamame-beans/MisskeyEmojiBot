# Misskey Emoji Bod
Misskeyの絵文字登録をDiscord上で申請できるDiscord用Botです！

## 機能
✅絵文字申請をボタンで簡単に作成
✅絵文字の各種情報を簡単に設定可能
✅絵文字の申請から承認・不認可、アップロードまでを自動化！

## コマンド
`/init`: 初期設定を行います。
`/ni_rilana`: このBotに関する情報を表示します


今後も機能を追加予定です！
